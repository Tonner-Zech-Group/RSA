Tutorials

Warning

All tutorials use simplified input settings to speed up the RSA simulations. For your product runs, carefully benchmark and motivate your settings!

Info

A complete and more extensive explanation of each input keyword is found in the "Input Keywords" part of this documentation starting with the introduction of Keyword Types.

Tip

As common for simulation softwares, it is highly advised to sort your simulations by several folders. If you want to run RSA simulations on a computing cluster, have a look at Tutorial 9 to learn how to write all information to a file for later evaluation.
